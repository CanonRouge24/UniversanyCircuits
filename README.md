# Installation

This pack was meant for Gregtech New Horizons in particular, but it should work with the "Dreamcraft" mod's "any circuits" in general.
Download the UniversanyCircuits.zip file and place it in your resourcepacks folder, then load it in game above GTNHRefreshed.
